## 🚀 &nbsp; What I've Shipped

> Not just projects — outcomes.

<br/>

### 📊 &nbsp; Financial Modeling Automation Suite
`Python · Bloomberg API · Excel · openpyxl`

**The problem:** Analyst memo prep took ~3 hrs of manual data pulls per idea.  
**What I built:** Python scripts pulling Bloomberg/CapIQ data → auto-structured sensitivity tables → clean Excel output.  
**The result:** Prep time cut to under 30 min per memo at PNTHR Funds.

→ [View Repo](https://github.com/vishanthashok/REPO_NAME_1)

---

### 🏢 &nbsp; AI Data Center Market Intelligence Dashboard
`Python · Power BI · SQL`

**The problem:** No centralized view of AI-driven data center demand across the comp set.  
**What I built:** Benchmarking tool aggregating 15+ public comps — hyperscaler capex trends, scenario-weighted demand forecasts, exec-ready Power BI dashboard.  
**The result:** Delivered as a primary research artifact at Evermount Partners.

→ [View Repo](https://github.com/vishanthashok/REPO_NAME_2)

---

### 🏗️ &nbsp; LBO Model Suite — Multifamily & Media
`Excel · VBA`

**The problem:** GPs needed institutional-quality LBO templates for lower-MM deal sizes.  
**What I built:** Full LBO models with integrated debt waterfall, IRR/MOIC sensitivity, GP/LP return splits — built for Fund Launch Partners.  
**The result:** Used live in due diligence across real deal pipelines.

→ [View Repo](https://github.com/vishanthashok/REPO_NAME_3)

---

### 📝 &nbsp; Investment Memo Generator
`Python · NLP · EDGAR API`

**The problem:** Memo drafting is repetitive — scraping filings, summarizing 10-Ks, formatting structure.  
**What I built:** Automated pipeline: EDGAR/news ingestion → NLP summarization → templated PDF/Word output.  
**The result:** Pitched 5 investment ideas to PNTHR partners using this exact workflow.

→ [View Repo](https://github.com/vishanthashok/REPO_NAME_4)
